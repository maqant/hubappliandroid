Product Blueprint Hub - Catalogue des Agents & Prompts
======================================================

Il y a bien 18 agents fixes définis dans le logiciel, ainsi que 2 agents dynamiques facultatifs (qui s'activent selon les besoins détectés dans le brief).

Les prompts ne sont pas stockés sous forme de fichiers statiques complexes, mais sont assemblés dynamiquement par le moteur d'exécution (MissionExecutor) à partir des définitions d'agents et des tâches planifiées.

Pour chaque tâche d'agent, la structure d'appel d'API se présente ainsi :
--------------------------------------------------------------------------------
SYSTEM PROMPT :
"You are agent [AGENT_ID]. Analyze and produce structured output for: [TASK_DESCRIPTION]"

USER PROMPT :
"Execute task: [TASK_NAME]
Description: [TASK_DESCRIPTION]
Agent: [AGENT_ID]"
--------------------------------------------------------------------------------

Voici la liste complète des 20 agents (18 fixes + 2 dynamiques) avec leurs rôles (Purpose) et la description de leurs tâches générant les prompts :
